Fungsi Persamaan Kuadrat

Pengertian dari fungsi kuadrat adalah sebuah fungsi polinomial atau suku banyak dimana pangkat tertinggi dari variable atau peubahnya adalah 2. Maka, bentuk umum dari fungsi kuadrat mendekati persamaan kuadrat.

Materi Matematika

Fungsi Kuadrat

Pengenalan Fungsi Kuadrat

Fungsi kuadrat berbentuk f(x) = ax² + bx + c, dengan a, b, c sebagai konstanta dan a ≠ 0. Fungsi ini membentuk grafik parabola pada bidang koordinat.

Sifat-Sifat

Grafik fungsi kuadrat memiliki puncak, sumbu simetri, dan dapat membuka ke atas atau ke bawah tergantung pada tanda konstanta a. Nilai diskriminan menentukan jumlah dan jenis akar fungsi.

Rumus

y = ax² + bx + c

dengan:

- a, b, dan c adalah koefisien konstanta (dengan a ≠ 0 ),

- x adalah variabel bebas.

- y adalah nilai fungsi.

Contoh Soal

Contoh Soal

Soal 1

Diketahui fungsi kuadrat f(x)= 2x² - 8x + 6. Tentukan koordinat titik potong dengan sumbu x dan sumbu y.

Jawaban:

- Titik potong dengan sumbu y:

substitusi x = 0

f(0) = 2(0)² - 8(0) + 6 = 6

Titik potongnya adalah (0,6)

- Titik potong dengan sumbu x:

2x² - 8x + 6 = 0 (dibagi 2)

x² - 4x + 3 = 0

Faktorkan:

(x-3)(x-1)=0

Titik potongnya adalah (3,0) dan (1,0)

Soal 2

Diketahui fungsi kuadrat f(x) = x² - 6x + 8. Tentukan koordinat titik puncak dan nilai minimum fungsi tersebut.

Jawaban:

- Koordinat titik puncak:

Gunakan rumus x = -b / 2a:

a = 1, b = -6:

x = -(-6) / 2(1) = 6 / 2 = 3

Substitusi x = 3 ke fungsi:

f(3) = (3)² - 6(3) + 8 = 9 - 18 + 8 = -1

Koordinat titik puncak adalah (3, -1).

- Nilai minimum:

Karena a > 0, fungsi memiliki nilai minimum di titik puncak:

Nilai minimum adalah -1.

Soal 3

Diketahui fungsi kuadrat f(x) = x² - 7x + 12. Tentukan akar-akar fungsi kuadrat tersebut.

Jawaban:

Faktorkan persamaan:

f(x) = x² - 7x + 12 = (x - 3)(x - 4)

Jadi akar-akarnya adalah

x = 3 dan x = 4.
